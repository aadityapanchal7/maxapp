# HAIRMAX NOTIFICATION ENGINE — Reference (authoritative for schedule + SMS copy)

## MODULE 1 — FINASTERIDE (ORAL / TOPICAL PATH)

### Dose insight (AI should know)

Community dose-response: **~0.5 mg ≈ 85–90% of DHT suppression vs 1 mg**. Many users stay on **0.5 mg** for better risk/benefit. Default to **tier target dose** but **allow user choice to remain on 0.5 mg** if they prefer.

### Side-effect ladder (check-in driven)

1. **First report:** Nocebo framing; ~43% vs ~15% in studies; give **2–4 more weeks** for adjustment.  
2. **Second report (2+ weeks later):** **0.25 mg daily** or **0.5 mg EOD**; many retain benefit at lower doses.  
3. **Third:** Topical fin (lower systemic) **or** pause **2 weeks** then restart **0.25 mg**.  

**Sexual sides specifically:** Resolve in **97%+** within **1–2 weeks** of stopping; if continuing → drop dose; fin isn’t the only path — adjust stack.

### Side-effect–sensitive onboarding

If **“had sides before”** or **“very concerned”:** skip oral fin; show **topical finasteride daily** (scalp at night; Hims combo / compounding). Lower systemic absorption.

### Daily notification

**1×/day** — time user-chosen or default **wake + 30–45 min** (not at exact wake unless merged with SkinMax AM block).

## MODULE 2 — MINOXIDIL

**2×/day** when on full stack.

| Slot | Time | Copy summary |
|------|------|--------------|
| **AM** | **Wake + 15 min** | Foam preferred AM; 1 ml / half cap; part to scalp; dry 15–20 min before styling; no wash **4+ h** |
| **PM** | **Bed − 90 min** | Liquid OK PM; 1 ml; dry **30–60 min** before bed / pillow |

**Budget tiers (framing):** Tier 1 **Prevention** — minox **optional**, don’t push. Tier **2–3** — recommend **2×/day**. Tier **4** — **strong** 2×/day; after **6+ mo** poor response → discuss **oral minoxidil** with derm (enzyme / sulfotransferase angle).

**Product bands (onboarding once):** Minimal — Kirkland liquid; Moderate — Kirkland PM + Rogaine foam AM; Flexible — foam AM+PM or topical fin/min combo.

**Rotating micro-tips (weekly):** dry scalp not wet; part to expose scalp; no blow-dry right after; coverage includes crown; foam if PG irritates; **wash hands after** (unwanted facial hair).

## TREATMENT RAMP (stagger modules)

- **Month 1:** Finasteride only (ramp **0.25 → target**). **Ketoconazole** shampoo starts immediately.  
- **Months 2–3:** Add **minoxidil 2×/day**.  
- **Month 4+:** Add **microneedling 1×/week**.  
- **Month 6+:** Major review — escalate options (fin 1 mg, microneedling, dut, oral minox, tretinoin + minox) per clinical judgment.  
- **Month 12+:** Stable stack; if still unhappy → **transplant** conversation with “**treatment continues after**” framing.
